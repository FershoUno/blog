# Debian 14 entra oficialmente en Freeze

El equipo de Debian ha anunciado oficialmente que Debian 14 (nombre en código `forky`) ha entrado en la fase de **Freeze** de paquetes, lo que significa que no se aceptarán nuevos paquetes a menos que sean correcciones críticas.

## ¿Qué significa el Freeze?

Cuando un lanzamiento de Debian entra en Freeze:

- **No se agregan nuevos paquetes** a menos que sean esenciales
- **Solo se permiten correcciones de bugs** críticos y de seguridad
- **Los equipos de pruebas** se enfocan en estabilizar la release

Esto es una señal clara de que el lanzamiento está cerca.

## Fechas clave

| Evento | Fecha |
|--------|-------|
| Freeze de paquetes | Julio 2026 |
| Freeze de transiciones | Agosto 2026 |
| Freeze de documentación | Septiembre 2026 |
| Lanzamiento estimado | Octubre 2026 |

## Novedades esperadas

Debian 14 incluirá:

- **Linux 6.12** como kernel predeterminado
- **GNOME 48** como escritorio predeterminado
- **Systemd 256** con nuevas funcionalidades
- **Soporte mejorado** para arquitecturas ARM64 y RISC-V
- **Instalador mejorado** con soporte para sistema de archivos Btrfs como opción

```bash
# Verificar versión actual de Debian
cat /etc/debian_version

# Actualizar a Debian 14 (cuando esté disponible)
sudo sed -i 's/bookworm/forky/g' /etc/apt/sources.list
sudo apt update && sudo apt upgrade -y
```

> Debian 14 promete ser una de las releases más sólidas de la historia del proyecto.

## Cómo ayudar

El equipo de Debian siempre necesita probadores. Si quieres ayudar:

1. Descarga la imagen de testing
2. Reporta bugs con `reportbug`
3. Únete a la lista de correo `debian-testers`

¡Cada bug reportado hace que Debian sea mejor!

# Guía de seguridad básica para servidores Linux

La seguridad de un servidor Linux no es opcional. Aquí tienes las medidas mínimas que deberías implementar.

## 1. Hardening de SSH

```bash
# Editar /etc/ssh/sshd_config
# Deshabilitar login como root
PermitRootLogin no

# Usar solo autenticación por llaves
PasswordAuthentication no
PubkeyAuthentication yes

# Cambiar puerto (opcional)
Port 2222

# Limitar usuarios
AllowUsers fernando admin

# Reiniciar SSH
sudo systemctl restart sshd
```

## 2. Firewall con iptables/nftables

```bash
# Usando nftables (moderno)
cat > /etc/nftables.conf << 'EOF'
table inet filter {
    chain input {
        type filter hook input priority 0;
        policy drop;

        # Permitir conexiones establecidas
        ct state established,related accept

        # Loopback
        iif lo accept

        # SSH
        tcp dport 2222 accept

        # HTTP/HTTPS
        tcp dport {80, 443} accept
    }

    chain forward {
        type filter hook forward priority 0;
        policy drop;
    }
}
EOF

sudo systemctl enable --now nftables
```

## 3. Fail2ban

```bash
# Instalar
sudo apt install fail2ban -y

# Configurar
cat > /etc/fail2ban/jail.local << 'EOF'
[sshd]
enabled = true
port = 2222
maxretry = 3
bantime = 3600
findtime = 600
EOF

sudo systemctl restart fail2ban
```

## 4. Actualizaciones automáticas

```bash
# Configurar unattended-upgrades
sudo apt install unattended-upgrades -y
sudo dpkg-reconfigure --priority=low unattended-upgrades
```

## 5. Monitoreo básico

| Herramienta | Propósito | Instalación |
|-------------|-----------|-------------|
| `fail2ban` | Anti-fuerza bruta | `apt install fail2ban` |
| `rkhunter` | Detección de rootkits | `apt install rkhunter` |
| `aide` | Integridad de archivos | `apt install aide` |
| `auditd` | Auditoría del sistema | `apt install auditd` |

## Checklist de seguridad

- [x] SSH con llaves únicamente
- [ ] Firewall configurado
- [x] Fail2ban activo
- [ ] Actualizaciones automáticas
- [x] Logs centralizados
- [ ] Backups automáticos
- [x] Monitoreo de integridad
